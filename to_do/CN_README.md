Milvus 是一款高速智能向量检索引擎，旨在通过以下方式帮助用户轻松应对海量非结构化数据挑战：

- 实现高性能、高精度、可扩展的海量特征向量检索。
- 实现对非结构化数据的近似查询和分析。
